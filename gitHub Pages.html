Create New Repo

  1. Open github
  2. Expand Code menu
  3. Copy URL
  4. Open Command Prompt
  5. Navigate to Directory: C:\>cd C:\Bootcamp\IntroToProgramming\MyPortfolio
  6. Type git clone + URL and hit Enter
  7. Type C:\>cd + repoName
  8. Type C:\>code . to open VSCode
  9. Copy URL

Create HTML document

  1. Add New File
  2. type doc + Enter
  3. Create h1 tag with some title
  4. Create p tag with some content
  5. Hit save

Commit Changes to gitHub

  1. Type Ctrl+` to open VSCode Terminal
    1. Type C:\...\CodeCollegePortfolio> git push origin main to confirm current branch
    2. Type C:\...\CodeCollegePortfolio> git add . to add ALL changes
    3. Type C:\...\CodeCollegePortfolio> git commit -m initialcommit to commit changes
    4. Type C:\...\CodeCollegePortfolio> git push origin main to add changes to Repo

Configure publishing source for GitHub Pages site

  1. Click Settings
  2. Click Pages from the Code and Automation section on the sidebar
  3. Go To Buid and Deployment
  4. Select Source = Deploy from Branch
  5. Select branch from the Branch dropdown menu
  6. Select Root from the Folder dropdown menu
  7. Hit Save

Checkout new branch and Make Changes

  1. Type C:\...\CodeCollegePortfolio> git checkout -b branchName [e.g. styling]
  2. Updates EXAMPLE:
    1. Add New CSS File called styles.css
    2. Go to index.css
    3. Type /link rel="stylesheet" href="styles.css"> to link styles.css
    4. Add styles to the styles.css and save file

Push Changes to gitHub

  1. Type Ctrl+` to open VSCode Terminal
    1. Type C:\...\CodeCollegePortfolio> git add . to add changes
    2. Type C:\...\CodeCollegePortfolio> git commit -m styling
    3. Type C:\...\CodeCollegePortfolio> git push origin branchName(e.g. styling) to push branch to gitHub
  2. Confirm that the branch is added in GitHub

pull Request to Merge Changes to Main branch

  1. Type Ctrl+` to open VSCode Terminal
    1. Type C:\...\CodeCollegePortfolio> git add . to add changes
    2. Type C:\...\CodeCollegePortfolio> git commit -m styling
    3. Type C:\...\CodeCollegePortfolio> git push origin branchName(e.g. styling) to push branch to gitHub
  2. Confirm that the branch is added in GitHub
